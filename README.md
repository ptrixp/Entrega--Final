## Instalación
Para la instalación del servidor se debe ejecutar el siguiente comando en la raiz del proyecto.
```bash
npm install -y
```
Para desplegar el proyecto
```bash
npm run dev //v1
npm start   //v2
```
instalar dependencia
```bash
npm init --y
```
